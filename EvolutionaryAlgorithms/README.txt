DESCRIPTION:
This program compares two evolutionary algorithms (Genetic Algorithm and Population Based Incremental Learning). We are testing each algorithm on MAXSAT problems.

HOW TO RUN:
The user must pass the following parameters as command line arguments.
File Name (string)		ex. t3pm3-5555.spn.cnf
Population (int)		ex. 100
Selection (string)		ex. ts
Crossover (string)		ex. 1c
Probability Crossover (double)	ex. .7
Probability Mutation (double)	ex. .01
Generations (int)		ex. 1000
Algorithm (string)		ex. g
disInterval (int)		ex. 100
